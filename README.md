------------------------------------------------------------
1. Bypassing Restrictions (Firewalls & Web Blocks)
------------------------------------------------------------

Networks at schools, workplaces, or public spaces often use
firewalls to block specific websites or throttle speeds.

Common techniques include:

1. VPNs (Virtual Private Networks)
----------------------------------
The most common method.
A VPN encrypts your data and routes it through an external server,
making it difficult for the local Wi-Fi router to see
what websites you are visiting.

Advantages:
- Encrypts traffic
- Hides browsing activity
- Bypasses geo restrictions
- Improves online privacy

2. DNS Tunneling
----------------
Encapsulating internet traffic within standard DNS queries
to bypass network firewalls.

Uses:
- Firewall evasion
- Data tunneling
- Restricted network access

3. Proxy Servers
----------------
Rerouting your web traffic through a third-party server
to mask your destination.

Benefits:
- Hides original IP
- Access blocked websites
- Faster access in some regions

------------------------------------------------------------
2. Bypassing Captive Portals
------------------------------------------------------------

Public Wi-Fi networks (airports, cafes, hotels)
often require users to accept terms or enter login credentials
on a web page before internet access is granted.

Methods sometimes discussed include:

1. MAC Address Spoofing
-----------------------
Temporarily changing your device's MAC address
to match a device that has already been authenticated
on the network.

2. DNS Manipulation
-------------------
Changing DNS settings on your device
to attempt bypassing login redirect pages.

------------------------------------------------------------
3. Connecting Without a Password
------------------------------------------------------------

Accessing a private or secure Wi-Fi network
without manually entering the security key.

1. WPS (Wi-Fi Protected Setup)
------------------------------
If physical access to the router is available,
pressing the WPS button or entering the WPS PIN
can allow devices to connect without typing
the Wi-Fi password.

2. Authentication Exploits
--------------------------
Advanced vulnerabilities in older Wi-Fi protocols
may sometimes allow attackers to bypass authentication
without knowing the password.

------------------------------------------------------------
4. Router Bypass Modes
------------------------------------------------------------

In networking hardware such as Starlink or ISP modems,
"Bypass Mode" is a legitimate feature used to disable
the router's internal Wi-Fi and routing functions
so a third-party router can be connected.

Common Uses:
- Better router performance
- Custom networking setups
- Advanced firewall management
- Gaming optimization
- Enterprise networking

------------------------------------------------------------
Security Recommendations
------------------------------------------------------------

To protect your Wi-Fi network:

- Use WPA3 encryption when available
- Disable unused WPS features
- Change default router passwords
- Monitor connected devices
- Update router firmware regularly
- Use strong DNS filtering
- Enable firewall protection
